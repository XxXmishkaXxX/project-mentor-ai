# US-11: Адаптация сайта под разные устройства

**Приоритет**: Should have
**Итерация**: 2
**Модуль**: frontend
**Backend**: —
**Frontend**: all components

## Формулировка

Как пользователь сайта, я хочу использовать ассистента с ноутбука, планшета или телефона, чтобы получать помощь в удобном формате.

## Критерии приёмки

- основные элементы сайта корректно отображаются на разных размерах экрана
- поле ввода и кнопка отправки доступны на мобильном устройстве
- ответ ассистента удобно читать с телефона
- таблицы или длинные блоки текста не ломают страницу
- пользователь может отправить новый вопрос без горизонтальной прокрутки

## Технические детали

Mobile-first CSS. Breakpoints: <768px (sidebar hidden, hamburger), 768–1024px (narrow sidebar), >1024px (full layout).

# US-02: Отправка вопроса ассистенту

**Приоритет**: Must have
**Итерация**: 1
**Модуль**: chat, rag
**Backend**: `backend/src/app/chat/controller.py` — POST `/api/chats/{chat_id}/messages` (SSE)
**Frontend**: `ChatInput.vue`, `ChatWindow.vue`, `stores/chat.js`, `api/chat.js`

## Формулировка

Как студент, я хочу ввести вопрос на сайте и отправить его ассистенту, чтобы быстро получить помощь по проектной деятельности.

## Критерии приёмки

- пользователь может ввести вопрос в текстовое поле
- кнопка отправки запроса доступна и понятна
- после отправки сайт показывает, что запрос обрабатывается
- ответ выводится на той же странице
- при пустом вопросе сайт просит ввести текст запроса

## Технические детали

ChatInput отправляет вопрос. Backend сохраняет user message в PG, запускает RAG pipeline, стримит через SSE. Frontend читает stream через fetch + ReadableStream.

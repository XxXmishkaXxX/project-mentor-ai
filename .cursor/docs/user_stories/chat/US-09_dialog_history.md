# US-09: История диалога в рамках сессии

**Приоритет**: Should have
**Итерация**: 2
**Модуль**: chat
**Backend**: `backend/src/app/chat/service.py`, `rag/pipeline.py`
**Frontend**: `ChatWindow.vue`, `ChatSidebar.vue`

## Формулировка

Как студент, я хочу видеть предыдущие вопросы и ответы на странице сайта, чтобы продолжать диалог без потери контекста.

## Критерии приёмки

- на странице отображается история переписки
- видны сообщения пользователя и ответы ассистента
- новые сообщения добавляются в конец диалога
- при уточняющем вопросе ассистент может учитывать контекст текущей сессии
- после обновления страницы поведение истории описано и понятно пользователю

## Технические детали

GET `/api/chats/{chat_id}/messages` возвращает историю. Pipeline включает последние N сообщений в промпт для контекста.

# US-13: Управление актуальностью базы знаний

**Приоритет**: Could have
**Итерация**: 3
**Модуль**: admin
**Backend**: `backend/src/app/admin/controller.py`, `admin/service.py`
**Frontend**: `DocumentList.vue`

## Формулировка

Как администратор базы знаний, я хочу обновлять или удалять устаревшие материалы, чтобы поддерживать качество и актуальность ответов.

## Критерии приёмки

- администратор видит список загруженных документов
- устаревший документ можно заменить новой версией
- документ можно удалить из базы знаний
- после изменения выполняется повторная индексация
- ассистент не должен использовать удаленные материалы

## Технические детали

DELETE removes from PG + Qdrant. Reindex: delete old chunks, re-parse, re-embed, re-upsert. Replace: upload new file for existing `document_id`.

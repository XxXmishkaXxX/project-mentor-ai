# US-01: Открытие главной страницы сайта

**Приоритет**: Must have
**Итерация**: 1
**Модуль**: auth, chat, frontend
**Backend**: `backend/src/app/auth/middleware.py` — session middleware
**Frontend**: `LoginPage.vue`, `RegisterPage.vue`, `ChatPage.vue`, `router/index.js`

## Формулировка

Как пользователь сайта, я хочу открыть главную страницу в браузере и сразу понять назначение сервиса, чтобы начать работу без установки дополнительных программ.

## Критерии приёмки

- сайт открывается в браузере без дополнительной установки
- на главной странице есть краткое описание ассистента
- пользователь видит поле для ввода вопроса
- на странице есть примеры вопросов, которые можно задать ассистенту
- интерфейс не перегружен лишними элементами

## Технические детали

Session middleware проверяет cookie. Vue Router guards редиректят неавторизованных на `/login`. ChatPage — основная страница с ExampleQuestions.

**Backend**: `backend/src/app/rag/pipeline.py`

## Формулировка

Как участник проектной команды, я хочу получить рекомендации по планированию задач, чтобы правильно распределить работу и сроки в команде.

## Критерии приёмки

- ассистент предлагает разбиение проекта на этапы
- ассистент объясняет, какие задачи можно выполнить сначала
- ответ может включать примерный план работ
- ассистент учитывает учебный формат проекта
- ответ помогает понять, что делать команде дальше

## Технические детали

Зависит от базы знаний (материалы по планированию, Agile, Scrum).

# US-04: Помощь в составлении технического задания

**Приоритет**: Must have
**Итерация**: 1
**Модуль**: rag
**Backend**: `backend/src/app/rag/pipeline.py` — quality of prompt and retrieval
**Frontend**: —

## Формулировка

Как студент, я хочу спросить у ассистента, как составить техническое задание, чтобы понять, какие разделы должны быть в документе.

## Критерии приёмки

- ассистент объясняет назначение технического задания
- ассистент перечисляет основные разделы технического задания
- ассистент может привести пример простой формулировки
- ответ подходит для учебного проекта
- ответ не содержит лишних сложных терминов без объяснения

## Технические детали

Зависит от качества базы знаний (должны быть документы про ТЗ) и system prompt (инструкция писать простым языком).

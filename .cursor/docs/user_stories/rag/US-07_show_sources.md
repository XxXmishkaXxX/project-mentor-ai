# US-07: Отображение источников ответа

**Приоритет**: Must have
**Итерация**: 1
**Модуль**: rag, chat
**Backend**: `backend/src/app/rag/pipeline.py` — возврат sources, `chat/service.py` — сохранение в `messages.sources`
**Frontend**: `SourcesPanel.vue`

## Формулировка

Как студент, я хочу видеть источники, на которых основан ответ, чтобы проверять информацию и больше доверять ассистенту.

## Критерии приёмки

- под ответом отображаются найденные источники
- для источника указано название документа или раздела
- источники относятся к заданному вопросу
- пользователь понимает, откуда взята информация
- если источников нет, сайт сообщает об этом отдельно

## Технические детали

RAG pipeline возвращает sources через SSE event. Sources сохраняются в `messages.sources` (JSONB). SourcesPanel — сворачиваемый блок под ответом.

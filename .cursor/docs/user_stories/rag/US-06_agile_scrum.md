## Формулировка

Как студент, я хочу задать вопрос по Agile, Scrum или DevOps, чтобы понять основные подходы к проектной деятельности.

## Критерии приёмки

- ассистент дает краткое объяснение выбранного подхода
- ассистент перечисляет основные роли, этапы или практики
- ответ связан с учебным проектом
- пользователь может задать уточняющий вопрос
- ассистент не уходит в неподходящие технические детали

## Технические детали

Требует контекста диалога (US-09) для уточняющих вопросов. База знаний должна содержать материалы по Agile/Scrum/DevOps.

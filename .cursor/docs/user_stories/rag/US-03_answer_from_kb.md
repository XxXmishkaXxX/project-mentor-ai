# US-03: Ответ на основе базы знаний

**Приоритет**: Must have
**Итерация**: 1
**Модуль**: rag
**Backend**: `backend/src/app/rag/pipeline.py`, `retriever.py`, `embedder.py`, `llm.py`
**Frontend**: `MessageBubble.vue`

## Формулировка

Как студент, я хочу получить ответ, сформированный на основе загруженных материалов, чтобы использовать проверенную информацию, а не случайный ответ.

## Критерии приёмки

- система ищет подходящие фрагменты в векторной базе данных
- ассистент формирует ответ с учетом найденных материалов
- ответ написан простым языком
- ответ соответствует вопросу пользователя
- если подходящие материалы найдены, они используются при генерации ответа

## Технические детали

Full RAG pipeline: embed query → search Qdrant top-5 → build prompt with context → stream LLM response.

# US-08: Честное сообщение при отсутствии информации

**Приоритет**: Must have
**Итерация**: 1
**Модуль**: rag
**Backend**: `backend/src/app/rag/pipeline.py` — score threshold check
**Frontend**: —

## Формулировка

Как студент, я хочу получить сообщение, если в базе знаний нет нужных данных, чтобы не использовать выдуманную информацию.

## Критерии приёмки

- если релевантные материалы не найдены, ассистент сообщает об этом
- ассистент не выдает неподтвержденный ответ как факт
- сайт предлагает переформулировать вопрос
- пользователь может отправить новый запрос
- сообщение написано понятно и без технических подробностей

## Технические детали

Если все scores от Qdrant < `RAG_SCORE_THRESHOLD` (0.7), pipeline возвращает стандартное сообщение. System prompt инструктирует LLM не выдумывать.

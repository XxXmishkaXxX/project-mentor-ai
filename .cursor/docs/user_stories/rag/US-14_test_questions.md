# US-14: Проверка ассистента на тестовых вопросах

**Приоритет**: Must have
**Итерация**: 1
**Модуль**: rag
**Backend**: `backend/tests/test_quality.py`, `backend/scripts/seed_knowledge_base.py`
**Frontend**: —

## Формулировка

Как разработчик, я хочу запустить проверку ассистента на наборе тестовых вопросов, чтобы оценить качество ответов перед демонстрацией проекта.

## Критерии приёмки

- подготовлен набор из 20 или более тестовых вопросов
- для каждого вопроса фиксируется ответ ассистента
- ответы проверяются на релевантность
- результаты проверки сохраняются в таблице или отчете
- по итогам проверки команда понимает, что нужно улучшить

## Технические детали

Скрипт прогоняет набор вопросов через pipeline, сохраняет ответы и sources, проверяет наличие источников и релевантность.

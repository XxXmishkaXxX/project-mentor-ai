## Описание

Стартовый каркас SPA: Vue 3 с Vite, точка входа приложения и базовый макет. Необходимые зависимости подключаются сразу (`vue`, `vue-router`, `pinia`, `axios` или нативный `fetch`) для последующих задач по HTTP и состоянию.

## Файлы

- `frontend/package.json` — зависимости и скрипты (`dev`, `build` и т.д.)
- `frontend/vite.config.js` — конфигурация Vite и алиасы
- `frontend/src/main.js` — монтирование приложения, плагины Router и Pinia
- `frontend/src/App.vue` — корневой компонент и `<router-view>`
- `frontend/index.html` — HTML-оболочка и точка входа для Vite

## Реализация

Инициализировать проект на Vue 3 + Vite. Установить и подключить `vue-router`, `pinia`, `axios` (или использовать `fetch` как единый HTTP-слой). В `main.js` зарегистрировать Pinia и Router. В `App.vue` разместить `<router-view />`. Структура каталогов `src/` готова к добавлению страниц и стора.

## Критерии готовности

- `npm run dev` запускает dev server на `http://localhost:5173`
- Приложение рендерится без ошибок в консоли

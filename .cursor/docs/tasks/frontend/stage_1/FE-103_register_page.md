# FE-103: RegisterPage

**Приоритет**: Must have
**Этап**: 1
**US**: US-01
**Зависимости**: FE-101
**Статус**: не начата

## Описание

Страница регистрации: username, email, пароль и подтверждение пароля. Валидация длины и совпадения паролей. Обработка дубликата пользов/email с бэкенда.

## Файлы

- `frontend/src/pages/RegisterPage.vue` — форма регистрации

## Реализация

Поля: username, email, password, confirm password. Валидация:

- username: 3–50 символов
- password: минимум 8 символов
- confirm password: совпадение с password

Кнопка отправки вызывает `authStore.register(...)`. По успеху → редирект на `/login` (или авто-логин, если так зафиксировано в проекте — по умолчанию `/login`). При ответе о дубликате (409 или текст ошибки) — показать сообщение. Ссылка на `/login` для уже зарегистрированных.

## Критерии готовности

- Все правила валидации работают на клиенте
- При успехе — переход на `/login` (или согласованное поведение)
- При дубликате/конфликте — отображается ошибка

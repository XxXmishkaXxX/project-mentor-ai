## Описание

Одно сообщение чата: различие ролей user / assistant по выравниванию и фону, отображение текста (при необходимости markdown), блок источников под ответом ассистента.

## Файлы

- `frontend/src/components/MessageBubble.vue` — презентация одного сообщения

## Реализация

**Props:** `message` — `{ role, content, sources?, created_at? }`.

User: выравнивание вправо (или иконка пользователя), светлый фон. Assistant: влево, другой фон. Контент: безопасный вывод текста; при наличии markdown в ТЗ — лёгкий парсер (например `marked` + DOMPurify по политике проекта). Если `sources` непустые — внизу `SourcesPanel`. Опционально краткая метка времени по `created_at`.

## Критерии готовности

- Сообщения пользователя и ассистента визуально различимы (справа / слева)
- Контент читаем и не ломает вёрстку
- Источники отображаются только у ответов ассистента при их наличии

## Описание

Главный экран после входа: боковая панель чатов фиксированной ширины, основная область с окном сообщений и полем ввода снизу. Шапка с информацией о пользователе и выходом. Пустой чат показывает примеры вопросов.

## Файлы

- `frontend/src/pages/ChatPage.vue` — композиция layout и дочерних компонентов

## Реализация

Layout: `ChatSidebar` слева (ширина ~250px), справа flex-колонка: сверху header (user + logout через `authStore`), ниже `ChatWindow`, внизу `ChatInput`. При отсутствии сообщений в текущем чате — показать `ExampleQuestions`. Синхронизация `route.params.id` с `chatStore.currentChatId` при необходимости.

## Критерии готовности

- Layout стабилен, компоненты связаны со store
- Пустой чат показывает приветствие и блок примеров вопросов

# FE-110: ExampleQuestions

**Приоритет**: Must have
**Этап**: 1
**US**: US-01
**Зависимости**: FE-104
**Статус**: не начата

## Описание

Набор карточек с типовыми вопросами для пустого чата. Клик создаёт чат при необходимости и отправляет выбранный вопрос.

## Файлы

- `frontend/src/components/ExampleQuestions.vue` — сетка/список карточек

## Реализация

Примеры текста:

- «Как составить техническое задание?»
- «Какие этапы у Scrum?»
- «Как спланировать работу команды?»
- «Что такое user story?»

Карточки кликабельны. По клику: если нет `currentChatId` — `createChat()`, затем `selectChat` и `sendMessage` с текстом вопроса. Компонент показывается только когда в текущем чате нет сообщений (проп или вычисление из store).

## Критерии готовности

- На пустом чате карточки видны
- Клик создаёт чат (если нужно) и отправляет вопрос
- При появлении сообщений блок скрывается

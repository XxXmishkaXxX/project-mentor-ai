**Приоритет**: Must have
**Этап**: 1
**US**: US-01
**Зависимости**: FE-104
**Статус**: не начата

## Описание

Боковая панель со списком чатов: создание нового чата, сортировка по дате обновления, выбор чата, удаление с подтверждением, визуальное выделение активного чата.

## Файлы

- `frontend/src/components/ChatSidebar.vue` — список чатов и действия

## Реализация

Сверху кнопка «Новый чат» → `createChat()` и выбор нового чата. Список из `chatStore.chats`, отсортированный по `updated_at` по убыванию. Элемент: усечённый title, клик → `selectChat(id)` и навигация на `/chat/:id` при необходимости. Кнопка удаления → `confirm()` или модальное подтверждение → `deleteChat(id)`. Активный чат подсвечен. Компактная вёрстка.

## Критерии готовности

- Чаты отсортированы по дате обновления
- Клик загружает и показывает сообщения выбранного чата
- Удаление только после подтверждения

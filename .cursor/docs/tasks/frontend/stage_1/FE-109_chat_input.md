# FE-109: ChatInput

**Приоритет**: Must have
**Этап**: 1
**US**: US-02
**Зависимости**: FE-104
**Статус**: не начата

## Описание

Поле ввода сообщения: многострочное с автоувеличением высоты, отправка по Enter, новая строка по Shift+Enter, кнопка отправки со стрелкой, блокировка на время стриминга.

## Файлы

- `frontend/src/components/ChatInput.vue` — textarea, кнопка, события

## Реализация

`<textarea>` с логикой авто-resize (рост по содержимому, `max-height` ~200px и внутренняя прокрутка). Кнопка отправки (иконка стрелки). Enter → `emit('send', content)` при непустом тексте; Shift+Enter → перенос строки. Пока `chatStore.isStreaming` — disabled для textarea и кнопки. После отправки — очистить поле и сбросить высоту. Не отправлять пустые или только пробельные строки.

## Критерии готовности

- Enter отправляет, Shift+Enter — новая строка
- Во время стриминга ввод и отправка заблокированы
- Пустые сообщения не уходят; поле очищается после отправки

**US**: US-07
**Зависимости**: -

## Описание

Свёртываемый блок источников RAG под ответом ассистента: количество, название документа, фрагмент текста, опционально score.

## Файлы

- `frontend/src/components/SourcesPanel.vue` — аккордеон/раскрывающийся список источников

## Реализация

**Props:** `sources` — массив `{ document_title, chunk_text, score? }`.

По умолчанию свёрнуто. Кнопка «Источники (N)» раскрывает список. Каждый элемент: жирный `document_title`, превью `chunk_text` с усечением и возможностью развернуть полный текст. `score` — опциональный индикатор релевантности (текст или визуальная шкала). Если `sources` пустой или отсутствует — компонент не рендерится (или скрыт).

## Критерии готовности

- Кнопка с корректным числом N
- Раскрытие показывает заголовок документа и текст чанка
- При отсутствии источников панель не видна

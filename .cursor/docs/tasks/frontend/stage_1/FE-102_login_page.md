# FE-102: LoginPage

**Приоритет**: Must have
**Этап**: 1
**US**: US-01
**Зависимости**: FE-101
**Статус**: не начата

## Описание

Страница входа: email и пароль, отправка в auth store, переход в чат при успехе, отображение ошибок API/валидации. Минималистичный чистый UI.

## Файлы

- `frontend/src/pages/LoginPage.vue` — форма входа и навигация

## Реализация

Форма: поля email, password; кнопка «Войти»; ссылка на `/register`. Валидация: обязательность полей перед отправкой. Вызов `authStore.login(email, password)`. Успех → `router.push('/chat')`. Ошибка → сообщение под формой (текст с backend или общее). Состояние загрузки на кнопке во время запроса.

## Критерии готовности

- Валидация обязательных полей
- При успешном логине — переход на `/chat`
- При ошибке — понятное сообщение пользователю

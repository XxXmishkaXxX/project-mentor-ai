## Описание

Единая дизайн-система на CSS: переменные цветов, типографика, сброс/нормализация, базовые стили интерактивных элементов. Светлая минималистичная тема.

## Файлы

- `frontend/src/assets/styles/main.css` — глобальные стили, импорт в `main.js`

## Реализация

CSS variables: `--color-primary`, `--color-secondary`, `--color-bg`, `--color-text`, `--color-border` и при необходимости оттенки. Шрифт: system-ui stack или Inter/Roboto (подключить по желанию через `@font-face` или CDN). Включить минимальный reset/normalize. Базовые правила для `body`, заголовков, `input`, `textarea`, `button`, ссылки. Отступы и радиусы согласованы. Импортировать `main.css` в точке входа приложения.

## Критерии готовности

- Палитра и шрифт единообразны по всему приложению
- Базовые элементы выглядят целостно без «сырых» дефолтов браузера
- Тема светлая, внешний вид чистый и сдержанный

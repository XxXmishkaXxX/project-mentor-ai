**Приоритет**: Should have
**Этап**: 2
**US**: US-12
**Зависимости**: FE-202

## Описание

Таблица документов базы знаний: название, имя файла, статус индексации с цветовыми бейджами, число чанков, дата, действие удаления. Пагинация через offset/limit.

## Файлы

- `frontend/src/components/DocumentList.vue` — таблица и пагинация

## Реализация

Колонки: Title, Filename, Status (badge), Chunks, Date, Actions. Цвета статусов: `indexed` — зелёный, `error` — красный, `processing` — жёлтый, `pending` — серый. Кнопка Delete с `confirm` или диалогом. Пагинация: элементы управления «вперёд/назад» или нумерация, параметры `offset`/`limit` в `fetchDocuments` (расширить store/api при необходимости). Пустое состояние — дружелюбное сообщение.

## Критерии готовности

- Таблица отображает данные из store с корректными бейджами
- Пагинация запрашивает следующие страницы
- Удаление только после подтверждения

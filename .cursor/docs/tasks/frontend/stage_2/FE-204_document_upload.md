# FE-204: DocumentUpload

**Приоритет**: Should have
**Этап**: 2
**US**: US-12
**Зависимости**: FE-202
**Статус**: не начата

## Описание

Форма загрузки документа в базу знаний: файл, обязательное название, опциональное описание, drag-and-drop, индикатор прогресса, подсказка по форматам.

## Файлы

- `frontend/src/components/DocumentUpload.vue` — UI загрузки и вызов store

## Реализация

`<input type="file" accept=".pdf,.docx,.md">` (или расширенный список по ТЗ). Поле title (required), textarea description. Кнопка «Загрузить». Зона drag-and-drop: обработка `drop` и `dragover`. При отправке — `adminStore.uploadDocument`. Индикатор прогресса (если axios поддерживает `onUploadProgress`, иначе спиннер на время запроса). Подпись о допустимых форматах. Ошибки — текст под формой. Успех — `fetchDocuments` или автообновление списка в store.

## Критерии готовности

- Можно выбрать файл кнопкой или перетаскиванием
- Виден индикатор загрузки
- Допустимые форматы явно указаны
- Ошибки и успех обрабатываются с обновлением списка

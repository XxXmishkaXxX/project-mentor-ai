## Описание

Пара иконок «лайк / дизлайк» под ответами ассистента для отправки обратной связи на backend. Повторный клик по той же кнопке снимает голос; смена кнопки меняет тип отзыва. Опционально модальное окно для комментария.

## Файлы

- `frontend/src/components/FeedbackButtons.vue` — UI и вызов API

## Реализация

**Props:** `messageId` (string).

Две кнопки-иконки (thumbs up / down). По клику: `POST /api/messages/{id}/feedback` с телом по контракту BE-202. После голоса подсветить активное состояние; повторный клик по активной — отмена или toggle согласно API. Опционально: открытие модалки для текстового комментария перед отправкой. Размещение под пузырём ассистента, ненавязчивые стили.

## Критерии готовности

- Кнопки отображаются под ответами ассистента (где подключено)
- Клик выполняет POST; повторный клик отменяет/заменяет feedback согласно логике backend
- Текущее состояние голоса визуально отражено

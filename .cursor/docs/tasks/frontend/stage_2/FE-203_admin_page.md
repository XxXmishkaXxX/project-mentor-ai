## Описание

Страница администратора для управления базой знаний: загрузка документов и таблица списка. Доступ только при `role=admin` (guard из FE-004).

## Файлы

- `frontend/src/pages/AdminPage.vue` — композиция DocumentUpload + DocumentList

## Реализация

Заголовок «Управление базой знаний». Сверху `DocumentUpload`, ниже `DocumentList`. Ссылка «Назад к чату» на `/chat`. При монтировании вызвать `adminStore.fetchDocuments()`. Убедиться, что маршрут `/admin` защищён guard’ом для admin.

## Критерии готовности

- Страница недоступна не-админам (редирект)
- Есть блок загрузки и список документов
- Навигация обратно в чат работает

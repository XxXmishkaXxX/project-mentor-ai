# FE-301: Управление документами (расширенное)

**Приоритет**: Could have
**Этап**: 3
**US**: US-13
**Зависимости**: BE-301, BE-302
**Статус**: не начата

## Описание

Расширение админского UI: переиндексация документа по кнопке и замена файла через выбор нового файла. Состояния загрузки на уровне строки таблицы.

## Файлы

- `frontend/src/components/DocumentList.vue` — новые кнопки действий и индикаторы в строках
- `frontend/src/api/admin.js` — `POST /api/admin/documents/{id}/reindex`, `PUT /api/admin/documents/{id}`

## Реализация

В каждой строке (или в меню действий): кнопка «Переиндексировать» → вызов reindex API, локальный `loading` для `id`. Кнопка «Заменить файл» → скрытый file input → при выборе `PUT /api/admin/documents/{id}` с телом по контракту (multipart или JSON + отдельная загрузка — согласно BE-302). После успеха обновить строку или перезагрузить список. Блокировать повторные клики на время запроса для той же строки.

## Критерии готовности

- Переиндексация вызывается с UI и отображает состояние загрузки
- Замена файла возможна через выбор файла и успешный PUT
- Ошибки показываются пользователю, список согласован с backend

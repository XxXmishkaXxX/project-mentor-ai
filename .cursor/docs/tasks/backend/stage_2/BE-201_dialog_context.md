# BE-201: Контекст диалога в RAG

**Приоритет**: Should have
**Этап**: 2
**US**: US-06, US-09
**Зависимости**: BE-108
**Статус**: не начата

## Описание

Передавать последние N сообщений чата в промпт LLM, чтобы ответы учитывали предыдущий диалог (уточнения, продолжение темы).

## Файлы

- `backend/src/app/rag/pipeline.py` — формирование блока истории в промпте
- `backend/src/app/chat/service.py` — передача последних 10 сообщений в `pipeline.ask`

## Реализация

- В `chat/service.py` передавать последние 10 сообщений как `chat_history` в `pipeline.ask()`.
- В `pipeline.py` вставлять историю между system prompt и текущим вопросом.
- Формат: чередование user / assistant сообщений, совместимый с `messages` для LLM.

## Критерии готовности

- Сценарий: «Расскажи про Scrum» → «А какие в нем роли?» — второй ответ использует контекст без повторного полного запроса темы в одном предложении

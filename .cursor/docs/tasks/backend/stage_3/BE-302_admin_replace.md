# BE-302: Admin — замена документа

**Приоритет**: Could have
**Этап**: 3
**US**: US-13
**Зависимости**: BE-301
**Статус**: не начата

## Описание

`PUT /api/admin/documents/{doc_id}` — загрузка нового файла на место существующего документа с полной перевёрсткой индекса.

## Файлы

- `backend/src/app/admin/controller.py` — multipart PUT
- `backend/src/app/admin/service.py` — удаление старого файла и чанков, сохранение нового, переиндексация

## Реализация

- Принять новый файл через multipart.
- Удалить старый файл с диска и старые точки в Qdrant.
- Сохранить новый файл, обновить `filename`, `file_path`, выполнить parse/chunk/embed/upsert.
- Обновить `chunk_count`, `updated_at` и прочие поля метаданных.

## Критерии готовности

- Старые чанки отсутствуют в Qdrant; новый файл проиндексирован
- Запись в PG отражает новые `filename`/`file_path` и актуальный `chunk_count`

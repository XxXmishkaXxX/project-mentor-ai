# BE-104: RAG — chunker

**Приоритет**: Must have
**Этап**: 1
**US**: US-03
**Зависимости**: BE-001
**Статус**: выполнена

## Описание

Парсинг файлов PDF, DOCX, MD и разбиение текста на чанки с перекрытием и метаданными для последующей индексации.

## Файлы

- `backend/src/app/rag/chunker.py` — диспетчер по расширению, сплиттер текста

## Реализация

- `parse_file(path)` — маршрутизация по расширению.
- `parse_pdf` — PyPDF2; `parse_docx` — python-docx; `parse_md` — чтение как текст.
- `split_into_chunks(text, chunk_size=512, overlap=64)` — разбиение по предложениям/абзацам с учётом лимитов (значения по умолчанию из `RAGConfig`).
- Возврат: список `{text: str, metadata: dict}`; в metadata — имя файла, `chunk_index`.

## Критерии готовности

- PDF, DOCX, MD корректно извлекаются в текст
- Чанки не превышают заданный размер; overlap между соседними чанками работает

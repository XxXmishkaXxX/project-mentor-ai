# BE-112: Скрипт загрузки базы знаний

**Приоритет**: Must have
**Этап**: 1
**US**: US-03, US-14
**Зависимости**: BE-104, BE-105, BE-106
**Статус**: выполнена

## Описание

Утилита для начальной загрузки документов из каталога `knowledge_base/` в Qdrant с записью статуса в PostgreSQL.

## Файлы

- `backend/app/entrypoints/seed_knowledge_base.py` — CLI-команда: обход файлов, chunk, embed, upsert, PG

## Реализация

- Чтение файлов из `knowledge_base/`.
- Для каждого файла: parse → chunk → batch embed → upsert в Qdrant → запись/обновление `documents` в PG со `status=indexed`.
- Идемпотентность: пропуск уже проиндексированных документов (по договорённости: hash пути/файла или флаг в БД).

## Критерии готовности

- После запуска документы доступны в векторном поиске
- Повторный запуск не ломает данные и не дублирует индекс без необходимости
